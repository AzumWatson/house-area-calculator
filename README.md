# 房屋面积计算器（Web）

一个纯前端的小工具，用来在画布上勾勒户型边线并计算面积。  
支持多个多边形、开放草图（未闭合线）、墙厚内面积、移动端手势、边长手动修改等功能。

项目由GPT 5.3 Codex辅助完成。

## 功能概览

- 画布点选成线，接近起点可自动闭合为多边形
- 支持多个多边形同时存在，并在图形内部显示面积标记
- 支持开放草图暂存与续画（点击端点继续）
- 正交模式（默认开启）：仅水平/垂直拉线
- 对齐辅助：
- 长度对齐（与已有边接近同长时吸附）
- 轴线对齐（对齐到水平/垂直边）
- 点投影对齐（对齐到已有点的 x/y）
- 可设置墙厚，并支持“只计算内面积（扣除墙厚）”（默认开启）
- 可直接点击边长标签修改长度，也可在侧栏线段列表中修改
- 删除模式下点击线段可删除，删除后保留缺口（不强制重新闭合）
- 缩放与平移：
- 桌面端滚轮缩放
- 移动端双指缩放
- `移动`工具拖动画板
- 快捷键：
- `Backspace` / `Ctrl+Z` / `Cmd+Z` 撤销
- `V` 添加点模式
- `M` 移动模式
- `X` 删除线模式
- `Esc` 关闭侧栏
- 设置本地持久化（`localStorage`），刷新后保留

## 运行方式

本项目不依赖打包工具，直接静态打开即可。

1. 直接双击 `index.html` 打开
2. 或使用本地静态服务器（推荐）：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 基本操作流程

1. 默认是`添加点`模式，点击画布开始画线
2. 继续点击添加点，形成草图
3. 贴近起点时点击可闭合为多边形
4. 如未闭合但想先切换工具，切到`移动`或`删除线`会自动暂存草图
5. 之后点击开放草图的端点，可继续沿该端点续画
6. 点击边长标签可直接输入长度（米）
7. 右侧侧栏可调比例（px/米）、墙厚、闭合阈值、面积模式等参数

## 工具与模式说明

- `添加点`：绘制与编辑主模式
- `移动`：拖动画板（配合缩放查看细节）
- `删除线`：点击线段删除，适合断开闭合图形或清理草图

## 设置项说明

- `正交模式`：限制新线段为水平/垂直
- `对齐已完成多边形`：是否把已完成多边形纳入对齐参考（默认关闭）
- `自动闭合阈值(px)`：靠近起点多少像素时触发闭合吸附
- `比例(px/米)`：像素与实际米制换算比例
- `墙厚(米)`：用于内轮廓面积计算
- `只计算内面积`：开启后展示内面积，关闭后展示外轮廓面积

## 移动端说明

- 画布全屏显示
- 右上角三横按钮打开右侧悬浮面板
- 面板打开时三横按钮隐藏，关闭后恢复
- 双指缩放，单指按当前工具模式操作

## 数据持久化范围

- 已持久化：绘图设置（正交、对齐开关、比例、墙厚、闭合阈值、面积模式）
- 未持久化：当前画布几何数据（点、线、多边形）和撤销栈

## 项目结构

- `index.html`：页面结构与控件
- `style.css`：布局与视觉样式
- `app.js`：核心绘图、对齐、面积计算、交互逻辑

## 已知限制

- 当前面积统计以“已闭合多边形”为单位；开放草图不计入面积
- 对齐为工程化启发式吸附，不是 CAD 级约束求解
- 未接入后端存储，跨设备不会同步
